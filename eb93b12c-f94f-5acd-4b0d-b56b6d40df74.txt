　　第507章
<br/>　　徐州，落凤城。
<br/>　　确切来说，落凤城已经不属于徐州。李乐天为了平衡各方面势力，得到诸家的支持，一口气封了十八家诸侯王。
<br/>　　李行哉封逍遥王，封地就在徐州落凤城。事实上，从长安城离开时，李行哉还是蹉跎满志的。毕竟，落凤城的情况虽糟糕，但和李行哉之前的经历比，已经算不错。
<br/>　　他流浪过天下各地，随着程大雷当了三年山贼，又在崔相手中当了大半年的傀儡。手下无有一兵一卒，所谓傀儡么，自然是别人指哪里，自己就得奔到那里。
<br/>　　但后来崔相将李行哉抛弃了，封地落凤城，虽然落凤城的基础盘很糟糕。但终究是自己的地盘，话再说过来，落凤城的基础盘再糟糕，能比青牛山蛤蟆寨的基础盘更糟糕么。程大雷刚出世的时候，可只有一匹瘦马，五个人。
<br/>　　现在不也搞得满天下风雨，提起程大雷的名字，虽然人人恨得咬牙切齿，但谁不是又怕又敬。
<br/>　　开局纵然糟糕些，但有地盘总比没地盘好，忍个几年，总能盘腾出一片天地。
<br/>　　也就是抱着这个念头，李行哉离了长安，奔向落凤城。路上还遭遇过几次刺杀，隐隐觉得，有人在暗中帮助自己，究竟是谁，却也查探不清楚。最后自己用了诈死一计，神不知鬼不觉的来到落凤城。
<br/>　　李行哉还颇满意自己的计策，有些沾沾自喜的意味。然而，等他到了落凤城，才知道情况是如何的糟糕。
<br/>　　城内账面上的人口是十万人，其实远没有这么多，仅有一些在册的居民，也是老弱妇孺居多。
<br/>　　那么壮年男子都去了哪里？
<br/>　　都去当了海盗山贼。
<br/>　　落凤城临海，时常找到海盗侵袭，城内的壮年男子，但凡能提得动刀的，要么是被山贼掳走，要么是被海盗掳走。
<br/>　　所以城内留下的，只有一些老弱病残。第一次入城的时候，李行哉牵着马在城中走，看到四璧凋零，街道两旁几乎没什么商铺。李行哉的第一感觉：这是一座死城。
<br/>　　一座城池最年轻的力量没了，就像一个男人被阉掉腰子，毫无生气，垂垂老矣，老到快死的地步了。
<br/>　　没有生气，这是落凤城的第一难。第二难就是海盗山贼做乱，时不时就上城过来枪一次。关键是落凤城已经穷到这种地步，还有什么好强的。
<br/>　　这两难已经令李行哉束手无策，然而还有第三难，便是落凤城的地理位置。落凤城位于徐州，又紧挨着扬州、兖州，是三方势力的夹缝。如果李行哉咬着牙忍了，估计没人会动他，可只要他稍微折腾出点动静。相府、将军府、扬州李星都不会看着他做大。
<br/>　　如此，就是李行哉面对的局面。
<br/>　　“逍遥王，逍遥王，逍遥你奶奶个大鸡腿……”
<br/>　　厨房内，李行哉骂咧咧出声，用斧子劈断一截干柴，丢进灶眼里。
<br/>　　他已经住进了城主府，现在也没有能力休憩改善，建一座王府。上一任城主被海盗杀了，城主府一个仆人也无。每天都得李行哉自己生火做饭，这种事李行哉倒也不是不会做，毕竟他一个人流浪过许多年。
<br/>　　柴米油盐酱醋茶，开门七件事，件件都要钱。而李行哉的账面上一分钱都无，现在都要他上山砍柴，借米下锅。
<br/>　　李行哉终于明白一件事，现在最要紧的不是发展问题，而是生存问题。或许某一天，自己就得饿死在城主府，呜呼哀哉，何其悲矣。
<br/>　　外面突然传来喊杀声，有人大呼小叫：“海盗来了，海盗来了。”
<br/>　　李行哉听到这动静，恶狠狠的骂了一声。右手拎着柴刀，想了想有拎起一根烧火棍，一瘸一拐出了城主府。
<br/>　　“天天抢，日日抢，把这里当你们家后花园么！”
<br/>　　他离了街上，见十几个海盗正在劫掠。海盗也并非动辄上千人，多的是七八个人，十几人的小团伙。因为落凤城只是一座小城，无兵可守，他们时时上案劫掠百姓，一些女子幼儿常被他们掳走。
<br/>　　如今十几个海盗在长街上肆无忌惮，百姓惶惶而逃，逃不及的，就被海盗一刀杀了，换来海盗的哈哈大笑。
<br/>　　看到这一幕，李行哉怒从心头起，恶向胆边生，口中喝道：“呔，你家李爷爷在此，尔等狼心狗肺之徒，竟敢杀我百姓，速速来我刀下送死。”
<br/>　　他拦在长街当中，将百姓护在身后。百姓们都知道落凤城来了一逍遥王，却也不当回事，今天还是第一次见着面。
<br/>　　李家的男儿有一个算一个，虽然脑袋常常拎不清，但从没有一个懦夫。
<br/>　　李行哉今日无兵可差，无将可派，手中只有一刀一棍，可面对十几员海盗，他又何惧之有。
<br/>　　提着刀与诸人砍杀，先用柴刀统翻了一个，然后引得海盗攻来，将李行哉围在打中。
<br/>　　李行哉以棍打远，以刀斩近。他倒也不是手无缚鸡之力的文弱书生，可功夫终究不如何，面对超过自己太多的敌人，实在没什么法子。
<br/>　　罢罢罢！
<br/>　　李行哉长叹口气，心底想着是，我今日死了便死了吧。为保百姓而死，传出去后，也让天下人知道，我李家没有孬种。
<br/>　　正这时候，从海盗身后杀出一人，年纪还小，约莫十八九岁，提着一杆铁枪，一路杀将过来。招式猛，下手狠，长枪不知捅翻了几多个，与李行哉合在一处，一起向外厮杀。
<br/>　　李行哉见这小将如此了得，心中有了底气，杀威更胜。二人联手，将这十几个海贼打的落荒而逃，又逃得慢的，被那小将一枪投出，洞穿了心脏。
<br/>　　李行哉擦掉额头的鲜血，上三眼下三眼的打量这小将，口中道：“多谢你了，今日若不是你，我还真不知如何是好。”
<br/>　　那小将也同样打量着李行哉，道：“阁下就是逍遥王李行哉么？”
<br/>　　“哈。”李行哉摆摆手：“丢人了，丢人了。对了你叫什么名字啊？”
<br/>　　这员小将突然跪到，口中道：“逍遥王在上，某家戚继光特来投靠。”
<br/>(本章完)
