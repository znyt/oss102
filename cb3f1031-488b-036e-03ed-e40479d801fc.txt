　　第723章
<br/>　　徐神机同刘发财踉跄站起来，腰酸背痛，骨头都快散架。
<br/>　　所有人的目光都落在这两人身上，程大雷恨不得找个地缝钻进去。徐神机目光转了一圈，忽然哈哈笑了一阵。
<br/>　　谁也不知道他笑什么，程大雷倒是多少明白一些。风雨未动雷先动，无论如何，先想办法压住场子再说。
<br/>　　徐神机的笑声嘎然而至，倒是让大家又是一楞。
<br/>　　“出门没看黄历么，太岁头上动土，想死想疯了。”
<br/>　　雷动抱肩坐在马上，面对这一幕笑而不语。
<br/>　　陈忠向前进一步，想要将程大雷几人拦回来。他们只不过是过路人，没必要惹来杀身之祸，他们也是不知道荆棘岭匪贼的厉害。
<br/>　　陈德山伸手拦住对方，他毕竟上了些年纪，有些见识在。既然程大雷三人敢出手，便必然有出手的自信。
<br/>　　不过，不管是徐神机还是刘发财都没有出手的想法，他们冲程大雷拱了拱手。
<br/>　　“请牛将军出手。”
<br/>　　程大雷一阵无语，无可奈何手搁在剑柄上，抬起头望着面前雷动一伙。
<br/>　　“给条活路吧，你们现在走还来得及，今天我不想开杀戒。”
<br/>　　“找死。”
<br/>　　李胖子冷哼一声，双手拎着大锤，照着程大雷脑袋砸来。这一下如果砸中，程大雷的脑袋估计会像西瓜一样被砸成稀巴烂。
<br/>　　程大雷撇撇嘴，一脚踹了出去。
<br/>　　李胖子被踹飞出去，武器也重重落在地上。程大雷这一脚踹在他的小腹上，一张脸瞬间变成黑色，估计下半辈子再也不能当男人了。
<br/>　　雷动眯着的眼睛睁开，李胖子势沉力猛，出手时也是空门大开，这是他的缺点。不过，知道这一点是一回事，能针对这缺点出手是另外一回事。刚才程大雷这一脚，角度刁钻，抬腿稳准狠。即便是雷动，自量也未必做不到。
<br/>　　“原来是个硬茬子，倒是我打了眼。”
<br/>　　程大雷深吸一口气，道：“如果你不想走，再晚怕就离不开此地了。”
<br/>　　人的名树的影，程大雷若报出真实姓名，估计雷动要吓得从马上摔下来。但程大雷顶着牛三斤的名字，还得靠自己双手双脚做事。
<br/>　　雷动从马上跃下来，拿起马背上的朴刀，冲程大雷亮了个刀架子。
<br/>　　“请了。”
<br/>　　程大雷不是很愿意搭理他，如这号人撑死了也就是为祸一方，但程大雷是祸一国的存在。但今天既然碰上了，没办法也只能赏脸与他动手。
<br/>　　雷动向程大雷攻来，招式也委实生猛，估计实力能和刘发财较量个高低。
<br/>　　程大雷也不想表现得太过匪夷所思，没有出剑，空手与他过招。三招之内，只听得咔嚓声响，程大雷捏断了他的手腕。
<br/>　　雷动抱着胳膊，痛得撕心裂肺，却咬牙强撑着。眼睛盯着程大雷，他必须重新认识程大雷，估计对方的实力。
<br/>　　“好汉，井水不犯河水，过江龙想压过地头蛇，管得太宽了吧？”
<br/>　　程大雷没有吭声，手压着剑柄，杀人这种事一旦杀顺了手，就会养成习惯。程大雷也是在遏制，今日实在不想把这里搞得血腥。
<br/>　　“青山不改，绿水长流，你有胆在这里等着。”
<br/>　　雷动一伙拖起李胖子，狼狈逃窜，程大雷也没有阻拦。当几人走了后，程大雷的手才算是松开剑柄，长长出了口气。
<br/>　　的确要遏制自己的杀心呐。
<br/>　　等他回过头来，却是吓了一跳。陈德山一躬到地，开口道：“多谢好汉出手相救，救了我陈家上下三百多口。”
<br/>　　“好说，好说。”徐神机拦在了陈德山面前：“我们都是这样行侠仗义的性子，路见不平自然就要拔刀相助。”
<br/>　　陈德山也是没有料到，他无意间收留的这几人，竟然解了陈家的灭顶之灾。感激之情溢于言表，他张罗仆人准备饭菜，在家里大摆宴席，轰散了来围观的庄客，将程大雷几人请在上座。
<br/>　　鱼洞真拎着崔白玉从屋顶上跃下来，虽然是女眷，但这荒野村庄也不讲究这些，于是两人也落了席。
<br/>　　刘发财与徐神机打开腮帮子，甩开后槽牙，这一路上没吃过什么好东西，全靠着今天补一补。
<br/>　　程大雷倒还稍微矜持些，与陈德山虚伪的客套着。这陈德山明显是摸不准程大雷的来路，仍在套程大雷的话。程大雷也是见过世面的人，自然不会被其三言两语套出来。
<br/>　　倒是问了陈德山一些话，家中有多少人口，陈家庄又靠什么为生，是否有官府前来征粮。
<br/>　　这话说到了陈德山的辛酸处，他喝了一杯酒，道：“牛将军既然问起，老汉便有什么说什么。这一大家子人看似风光，内里却是苦不堪言。山贼官府经常过来劫掠，那家也不敢得罪。庄子里三百多人，开口便是要饭吃。也是强撑着而已，其中辛苦那是一句话两句话说得清楚。”
<br/>　　说着，陈德山拍了拍手，老仆陈忠领着两个手下抬着一口箱子过来。放在搁手桌上打开，箱子内是一尊黄灿灿的金佛。
<br/>　　程大雷倒抽一口凉气：“陈庄主，这是何意？”
<br/>　　陈德山道：“这尊六手佛是我为小女求来，今日牛将军对我陈家上下有活命之恩，这尊金佛双手奉上，谢过牛将军的救命之恩。”
<br/>　　徐神机和刘发财都凑过去脑袋，舍不得移开目光。崔白玉倒是无动于衷，心中还鄙夷他们没见过世面。
<br/>　　崔白玉什么奇珍异宝没见过，都是天上有地上无的宝物。至于这金佛，只要有钱就能打造，又算的了什么。
<br/>　　“这，这……”程大雷啧啧摇头。
<br/>　　“牛将军万万不可推辞。”陈德山坐直身子：“老朽活了大半辈子，只做了一件事，从来不欠人情。人让我一尺，我便让人一丈，滴水之恩，当涌泉相报。”
<br/>　　刘发财不由得对这老汉肃然起敬，纵然是山野村夫，但做人做事也是光明磊落。
<br/>　　“你误会啦。”程大雷推了推手，道：“我们还要赶远路，这委实有些重啊。”
<br/>　　陈德山愣了好大一回，试探道：“要不我给送到府上？”
