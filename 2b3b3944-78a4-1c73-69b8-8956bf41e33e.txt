　　第744章
<br/>　　“来，再来！”
<br/>　　大内校武场上，马超发出一声声怒吼。他手擎一杆木枪，在他面前已经砸倒了四五名鱼龙卫的精锐。
<br/>　　这些人不是被木枪砸断胳膊，便是小腿骨折。现在马超仍在找人出战，可已没人敢做他的对手。
<br/>　　对于鱼龙卫这位新上任的指挥使，大家从前只知道他有些楞，很仗义，不如何守规矩。现在大家才了解，这位指挥使还很暴力。
<br/>　　有小黄门领着李乐天来到此地，鱼龙卫窥见他出现，急忙跪下来行礼。
<br/>　　哗啦啦跪倒一片，只有马超呼哧呼哧喘着粗气，依旧立在演武场的正当中。
<br/>　　有人急忙向马超使眼色，但马超视若无睹，恍然没有看见李乐天出现。
<br/>　　有这样蛮横的臣子，偏偏就有李乐天这样大度的君王。他并没有生气，挥挥手让除马超以外的人退下。
<br/>　　对于这位突然出现的蛮将，李乐天还是偏爱，对于他偶尔的不守规矩，不仅不生气，反而生出几分亲近。
<br/>　　马超为何如此暴力，原因李乐天也明白，无非就是败在程大雷手中，听了几句风凉话，自己的兵器还丢了。
<br/>　　所以回到长安城后，马超便如疯了一般。疯狂找鱼龙卫中的高手比武，夜以继日的锤炼武艺，目的是能在程大雷面前找回场子。
<br/>　　李乐天听到这个消息后，不仅不生气，反而有几分高兴。据说马超虽然败了，但也差了一招半招而已。李乐天早知道马超很强，但他不知道马超竟然强到这种程度，原来已经能和程大雷论个胜负。
<br/>　　至于他败给程大雷，那岂不是很正常的事情么。
<br/>　　眼见马超还在生气，他倒是有个好强的性子。这对一个年轻人来说，也未必不是好事。
<br/>　　“败给程大雷也并不是很难接受的事，你要知道，许多人想在他面前求一败而不能。”
<br/>　　马超喘着粗气：“我不是其他人。”
<br/>　　李乐天上下打量着他，眼神中有几分欣赏。
<br/>　　“自从青牛山出世，他历经大小百余战，方才能有今日本事。你想要胜过他，这是好事，但还需要时间。”
<br/>　　“我会打败他的，很快，很快。”马超咬着牙道，任谁也能听出他声音中的坚定。
<br/>　　李乐天深吸一口气，道：“朕有一招枪法，你想不想学？”
<br/>　　马超看了李乐天一眼，虽然没有开口，但眼神中多少有些鄙视。
<br/>　　他成为李乐天的手下，可不是因为李乐天的武艺，而是因为对方的身份。
<br/>　　李乐天读懂了这鄙视，他负手在身后，呵呵笑了一声。曾几何时，自己也如马孟起一般，要做那天下第一，会天下英雄，收拾旧山河。
<br/>　　不过随着年岁渐增，风流总被雨打风吹去。
<br/>　　现在的李乐天已经不敢这么想了，他能做的，只不过是在有限的范围内完成一些事情。
<br/>　　但一个少年有这样的想法，总是好事。
<br/>　　长长叹了一声，李乐天缓缓叹道：“李家打天下，并没有半分侥幸。当初天下谁人不怕李家军，谁人不知李家盘龙枪。”
<br/>　　“盘龙枪？”
<br/>　　“嗯，听说过？”
<br/>　　“没有。”
<br/>　　李乐天撇撇嘴：“盘龙八打枪，是李家家传枪法，一共八招，当初太祖爷用这八招枪法，便打下了帝国十三州一百单八城，向来是……”
<br/>　　有句话李乐天没有说，这盘龙八打枪向来是传男不传女，只传李姓人。
<br/>　　但帝国走到今日，有些规矩似乎已不用守了。李家后继无人，若李家枪能在马超手中大放光辉也未必不是一件好事。
<br/>　　他走到兵器架前抽出一杆木枪，双手握枪，轻轻一甩，整杆枪便同时震荡，力道从枪身一直传到枪尖。
<br/>　　“来。”
<br/>　　……
<br/>　　程大雷已经到了长安城，直到此时，方才算是松了口气。至少到了长安城，便没人会请大兵围杀自己，自己也算是能过几天安生日子。
<br/>　　前些日子，长安城内风声鹤唳，李乐天针对相府的势力来了一场大清洗。在这种局势下，程大雷安插在长安城的探子也不得不撤回去一批。
<br/>　　但到今日，已经看不出曾经的肃杀，长安街道又热闹起来。这次藩王入京，各个茶楼饭庄都已做好准备，想趁着这次机会发一笔横财。而青楼船舫，也等待着恩客上门。
<br/>　　当然，真正有身份有实力的藩王，随行都携带着娇妻美眷，偶尔去青楼遣兴还行，他们也不至于混到要到青楼找女人。
<br/>　　程大雷要先到礼部报个到，之后他的住宿自有朝廷安排。还未靠近礼部大门，就见一身着官袍的人立在大堂门口。
<br/>　　来往官员，见到他都纷纷上前行礼，有些自觉身份低不够格的，也不敢太冒失，只是远远绕开。
<br/>　　这人自然就是从龙有功的百里胜，谁也知道他现在正是如日中天，未来很有可能继承尉迟离的地位，长安城再起一座将军府。
<br/>　　可是，他今日为何会出现在礼部。
<br/>　　程大雷脚步不自觉的停下，这人该不会是来报仇的。他当初如丧家犬一般逃出凉州，说来说去和自己也有些牵扯。当真是时来天地皆同力，运去英雄不自由。谁能想到百里胜又抖了起来。
<br/>　　程大雷深吸一口气，大步向前，人未到声先至。
<br/>　　“百里侯爷。”
<br/>　　百里胜还没回过神来，已经被程大雷攥住双手，程大雷声音哽咽，眼眶湿润。
<br/>　　“侯爷可是让我好想，这些年，侯爷吃苦了。”
<br/>　　百里胜当真就是在这里守着程大雷，他自然不会向程大雷下黑手，只不过，心里多少有些怨气，想在程大雷面前抖索抖索威风。
<br/>　　谁料程大雷来了这一处，他没反应过来，只是『嗯嗯』的应承着。
<br/>　　“想当初我在长安，是谁与我把酒言欢，是谁和我促膝长谈。”程大雷紧握百里胜双手：“都是侯爷您呐。”
<br/>　　百里胜本来是官场的老油条，只是在军中历练一番，人已经变得沉稳，有些不太适应程大雷这一套。
<br/>　　来往官员看见这一幕，心中暗暗感慨：百里侯爷和程大雷的感情果然深厚呐。
<br/>(本章完)
