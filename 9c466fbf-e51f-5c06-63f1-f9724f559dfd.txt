　　第281章
<br/>　　窦竹仝的山寨内，阴云弥补，众人心情都不太好。
<br/>　　“那男人在干什么，还抱着死人号丧呢？”花皮虎。
<br/>　　“大哥回来后就在屋里不出来，送进去的饭菜他也没动。”喽啰忐忑道。
<br/>　　自从上次事情后，窦竹仝闭门不出，只陪着吕春花的尸体茶米不进。
<br/>　　花皮虎推开了门，屋外的光线透进来，一股腌脏气味弥漫在房间内。窦竹仝一身黑衣坐在地上，床上摆着吕春花的尸体，安静祥和，宛若睡着了一般。
<br/>　　花皮虎叹了口气，道：“痴汉子，我知道你心里恨我，以为我仗势欺你，便偷偷在外面养着小不与我说。可我们是结发夫妻，斩不断的情分，你有什么话不能和我说。如果你肯对我说一声，又何苦她在外面受罪。可你不肯说，我也赌这一口气，哎，也是怪我，不然也不会害她如此下场。”
<br/>　　几日不见，窦竹仝宛若三魂七魄去了大半，吕春花是死了，他和一个活死人也差不了多少。
<br/>　　“可她落在山贼手中，不知已被怎样糟蹋，我杀了她也是为了全她的名声。你恨我怨我，我都不怪你，可害她死的是那帮山贼，你若是个男人，就该替她报仇，斩了程大雷的脑袋祭奠，之后将她风光大葬，入你窦家的祖坟，如此让她死了也有个名份。”
<br/>　　“我会亲手杀了程大雷。”窦竹仝终于开口，声音沙哑。
<br/>　　“这才不愧是我的男人。”花皮虎道：“我已经请了其他寨主来这里议事，十八家势力一起联手，凭蛤蟆寨哪点儿人，一夜之间就可以血洗他们。这，才是你现在应该做的。”
<br/>　　“他们……都会来？”
<br/>　　“当然，云中龙的人，祁珲坡，朱五苍等等这些人都会来，我表哥也会派人来帮我们。现在他们马上就要到了，你打算就这个样子去见他们？”
<br/>　　窦竹仝咬咬牙，从地上站起来，骨节发出吱吱嘎嘎的响声。
<br/>　　“来人，给我准备热水。”
<br/>　　窦竹仝重新振作起来，让手下人烧水洗澡，沐浴更衣。
<br/>　　“来人，来人……”花皮虎用手招呼着：“把这贱人的尸体抬出去烧了，屋里的东西也一块烧干净。”
<br/>　　窦竹仝梳洗完毕，又恢复了几分精气神，他回到屋子，却看到屋子空空如也，不仅吕春花的尸体不见了，被褥碗碟等物也都被搬走了。
<br/>　　“人，人呢……”窦竹仝结结巴巴。
<br/>　　“大哥，是大嫂让我们清出去的，大哥……”
<br/>　　窦竹仝感觉一口热血往前涌，身体晃晃悠悠，两眼一黑闷头载倒。
<br/>　　……
<br/>　　黄昏，青叶镇。
<br/>　　一队凶神恶煞的山寨快马奔来，打破了小镇傍晚的平静。
<br/>　　“诸位老少爷们都听好了，我们是蛤蟆岛蛤蟆寨蛤蟆大王的人。”高飞豹在马上大叫：“现在蛤蟆大王要你们镇的男人去干活，听话的好吃好喝还有银子拿，若敢嘣半个不字，别怪二爷爷的刀不认得你们！”
<br/>　　小镇一时间鸡飞狗跳，小孩叫，大人哭，乱做一团。
<br/>　　“各位好汉，您行行好，我们都是上有老下有小。”一个白发苍苍的老者道。
<br/>　　“蛤蟆大王不会让你们白干活，谁去了都有银子拿。”
<br/>　　老者心里有苦说不出，这要是被山贼抢去了，那还有什么活路，唯一的结果只能是死路一条。
<br/>　　“大王爷爷，求求您大恩大德，饶了我们吧。”
<br/>　　“我是在和你商量嘛，我是在抢人懂不懂。”高飞豹暴跳如雷：“弟兄们，开始抢人！”
<br/>　　一队人穿堂入室，把镇上的男丁都捞了出来。
<br/>　　“狗贼，老子和你们拼了。”一个大汉大吼道。
<br/>　　“孩他爹，你不要玩命，孩他爹……”一个妇人死死拦住他。
<br/>　　高飞豹轻而易举将他砸倒，口中挥挥手道：“带走，带走。”
<br/>　　来去如风，最后从镇上搜刮出来一二百人，聚在镇中央的广场。
<br/>　　“蛤蟆大王只是要你们去干点活，你们要是不老实的，想想你们妻儿老小的命。”高飞豹骑在马上喊道。
<br/>　　一众汉子咬牙切齿，只想和这帮山贼拼命，但想想上有老母，下有妻儿，顾及这帮山贼的恐怖，他们只能将心中的火气压下去。
<br/>　　“带走。”
<br/>　　高飞豹挥挥手，带着这一二百壮劳力离开。在他们走后，青叶镇哭声一片。
<br/>　　“镇长，他们还能活着回来么？”
<br/>　　“会回来的，只是山贼抓他们去干活……”
<br/>　　“嘿，还回来什么，难道忘了琴川关抓走的人，最后一根骨头都没见到，活不见人死不见尸的。”
<br/>　　“老张，你儿子今天在外面，没被抓走你很高兴是不是！”
<br/>　　高飞豹押着人回到岛上，岛上这时候已经有超过两千人。这些人都是程大雷派人从附近村镇抢回来的，现在都聚在岛上。
<br/>　　“大当家，我回来了，咱们明天去哪儿抢人？”高飞豹。
<br/>　　“呃……人已经差不多够了，明天不用去抢了，让他们干活吧。”
<br/>　　“我知道啦，大当家，我今天还听到一个消息，琴川关莫鸣米也去抢过人，据说这些人被抢到琴川关后，就再也没有回来过。”
<br/>　　“这件事我也听说了，血汗工厂么。”
<br/>　　两千壮汉聚在一起，他们大多是庄稼汉，即便其中有凶悍之辈，可落到山贼手中心里却还是打鼓。
<br/>　　“木匠，有没有会做木匠的，往前走两步。”徐神机。
<br/>　　“瓦匠，瓦间往前走走两步。”
<br/>　　这些人摸不准山贼的路数，是直接杀死灭口啊，还是说要拉大家入伙成为山贼。如今，也只能按部就班的听山贼吩咐。
<br/>　　时间紧，任务重，只有让大家连夜干活，岛上到处点着篝火，照得夜空灯火通明。
<br/>　　“三哥，你说这帮山贼抓我们来做什么？”两个同村的男人，一边砌墙一边问。
<br/>　　“这不是让我们干活么？”
<br/>　　“关键是我们干完活后，还能让我们活着会去么？”
<br/>　　“这个……恐怕只有干完活后才知道。”
<br/>　　“那么咱们也别给他好好干，随便凑活凑活得了。”
<br/>　　二人打着同样主意，可说也奇怪，二人砌过的墙又平又直，即便以前认真，也出不了这么好的活。
