　　第406章
<br/>　　程大雷查看崭新的系统界面。从山寨升级到城池，开启攻城模式后，系统界面发生了翻天覆地的变化。
<br/>　　现在他脑海中，出现蛤蟆城的微缩模型，里面有蛤蟆城的各种建筑。如兵营、矿场、工匠坊……，当程大雷把注意力集中在上面时，就可以看到每样建筑的详细情况。
<br/>　　程大雷忽然发现，本来以为消失的系统技能，其实不能说完全消失。比如医馆，在医馆养伤的恢复率提升，这便等于代替了治愈技能。而城墙的坚固程度，可以在受到损坏后迅速恢复，这等于原先的固防功能。
<br/>　　现在的蛤蟆城只是初级城池，相比山寨当然可以称得上兵强马壮，但对于城池而言，只不过是起步阶段。
<br/>　　目前的蛤蟆城，共有十万人口，战斗人员大概在三万左右，其余是后勤和平民。山寨已经可以满足自给自足，这两年风调雨顺，蛤蟆城的粮仓爆满。而每年来往此地的商队，也能给蛤蟆城带来不错的税收。
<br/>　　现在的程大雷完全配得上琴川之主这个称号，从前有人若攻打蛤蟆城，程大雷死也能往对方身上咬下一口肉，其实还是玉石俱焚，亡命徒的打法。不过现在蛤蟆城已经要人有人，要粮有粮。不谦虚来说，目前帝国任何一股势力来打蛤蟆城，程大雷都有信心自保。当然，现在还没到攻不破的地步。
<br/>　　至少，程大雷很满意如今山寨的发展节奏。
<br/>　　程大雷打开了系统商店，原先系统商店出售的商品有：皮甲、青苓散、连击驽，火药。随着山寨升级，系统商店自然而然有了新的商品出售。
<br/>　　黑鬃马：优秀战马，售价二十万恐惧值，耐力优秀，适合长途行军。
<br/>　　战马在如今这个时代，是极重要的战略资源。琴川关虽然和戎族接壤，是帝国的产马区。程大雷还特意将落玉寨开垦为马场，以训练骑兵。但即便如此，蛤蟆城三万人的军队，骑兵数量不过才三千而已。
<br/>　　这支骑兵队伍，由赵子龙指挥，就是蛤蟆城的龙字军，同时也是蛤蟆城内人数最少的一支部队。可毫无疑问，他们是蛤蟆城战斗力最强的那支队伍，也是消耗资源最多的。
<br/>　　毕竟，养一匹马要比养一个兵贵多了。
<br/>　　程大雷很想买匹黑鬃马，看由系统提供的战马品质如何。但进去看了看，留下一笔进店费后，又灰溜溜的离开了。
<br/>　　好久没打仗了，随着一直消耗，程大雷目前的恐惧值所剩无几。暂时还没有奢侈到用二十万恐惧值买一匹黑鬃马的地步。
<br/>　　目前系统商店出售的商品，连击驽山寨已经可以仿制，火药山寨自己也可以生产。火药的配方进步，还要感谢炼金术师希曼。
<br/>　　炼金术师希曼，生命女神的坚定信仰者，虽然在追求真理的道路上撞得头破血流。但对于各种物质的属性，他还是比较了解的。随着他的到来，火药的配方被改良，如今已经不弱于从系统商店购买的。
<br/>　　程大雷差人把刘悲和秦蛮叫过来，将系统奖励的战旗交给刘悲，由他发给手下的指挥官。这战旗是附带系统技能的，可以提升己方士气，摧毁敌人的战斗意志。
<br/>　　之后，程大雷把秦蛮留下了。
<br/>　　秦蛮率领的蛮字军，人数在五千左右，军队作风和秦蛮一样，朴实无华，却又直接狠辣。
<br/>　　当然，这都是秦蛮的优点，而秦蛮的缺点是……他的脑子似乎不是特别好用。
<br/>　　陷阵冲锋，他是没什么问题的，但具体安排作战，他和关张赵三人有很大距离，甚至连高飞豹都比不了。
<br/>　　这只能说是个人天赋不同，可能秦蛮根本没开这方面天赋树。
<br/>　　程大雷将秦蛮留下来，是准备将梅花阵传授给他，由蛮字军训练，熟悉梅花阵的作战方式。
<br/>　　系统奖励下来的东西还剩下两样，一是梅花阵，一是提升绝世以下任意一人的等级。梅花阵程大雷准备交给秦蛮，他本身的本事就是梅花枪，这一看就是配套的么。
<br/>　　至于提升谁的等级？程大雷准备用在樊梨花身上。好东西，毕竟是要留给自己……的女人。
<br/>　　梅花阵并不复杂，十人一组，互相配合，可以合在一起，也可随时分开。当然，这需要极高的默契。
<br/>　　秦蛮沉默消化着程大雷的话，道：“关键在配合，也同样在指挥官身上。这大阵的阵眼在于临阵指挥上，能够分清形势，何时合，何时分，都必须把握正确判断。”
<br/>　　程大雷无语的看着秦蛮，秦蛮并不是笨，对于武学上的事情，他一眼就能看出端倪，这种两军对垒，他甚至都称得上专家。可也仅限于此了，排兵布阵，虚实进攻，他几乎一窍不通。
<br/>　　程大雷不得不承认一个事实，秦蛮有将才，而无帅才。
<br/>　　秦蛮注意到程大雷看自己的眼神，顿了顿道：“大当家，我说得不对。”
<br/>　　“对，没什么不对的。”程大雷看着秦蛮，道：“秦大哥，你以后要怎么活下去啊。”
<br/>　　秦蛮挠挠头，憨憨笑笑。虽然在山寨，大家都以程大雷为主，可无人时，程大雷对秦蛮还是这样称呼。
<br/>　　“反正有你在，终究是错不了的。我知道我脑子笨，有些事情也想不明白，你让我做什么，我便做什么。”
<br/>　　程大雷默然，几人都是从青牛山那座破山寨走来的。现在地位水涨船高，可身上的胆子也更加重了，同时日后也将会面临更大的危险。
<br/>　　程大雷忽然坐直了身子，道：“秦大哥，你可知我一身本事是从何处来的？”
<br/>　　“外人不是都说是神仙梦中传艺么，至于我，是不大相信的，总觉得鬼神之说，不太能当真。”
<br/>　　“不，传言不是假的。”程大雷坚定道：“我一身本事，的确都是梦中仙人所授，因为有仙人暗中保佑，我一路才可以顺风顺水，打下这偌大家业。”
<br/>　　程大雷深吸一口气：“昨夜我又做了一个梦，仙人对我提起了你。”
<br/>　　“啊！”秦蛮眼睛一亮：“仙人还知道我？”
<br/>　　“老是提起呢。”
<br/>(本章完)
