　　第330章
<br/>　　程大雷到这个世界的第一天，见过的人屈指可数：徐神机、徐灵儿、林少羽、秦蛮……都是一路走过来的弟兄，互相已十分了解。
<br/>　　从林少羽对程大雷说出那番话的时候，程大雷便明白，蛤蟆寨留不住他，心中的野望已经生根发芽，怎能轻易铲除。
<br/>　　“坐？”程大雷指了指旁边的位置。
<br/>　　林少羽默然垂手立在程大雷面前，摇摇头道：“大当家，我站着说话就行。”
<br/>　　程大雷倒也不坚持，轻抿口酒，随之将酒杯放下。
<br/>　　“人各有志，你要走我不留你。只是一日是兄弟，一世是兄弟，你这一去，风里雨里，是福是祸，谁也说不好，我这心里也难免牵挂。”
<br/>　　“大当家，日后若有用得着少羽的，不管我在那里，只要你一声令，水里水里到，火里火里到。”
<br/>　　“这话也是我要对你说的，竟让你先说出口。以后不管去了哪里，都记得是我兄弟，真有过不去的坎儿，记得传句话来，别自己硬撑着。”
<br/>　　“嗯。”林少羽重重的点点头。
<br/>　　“好啦。”程大雷道：“临别之前，送你们点东西，别两手空空的上路，回头到处说我做人小气。”
<br/>　　说罢，程大雷拍拍手，银眸牵着两匹马来，盔甲袍子也备了两幅。程大雷摊开一面战旗，上书斗大一个林字，递到林少羽手中。
<br/>　　“这……好像是关二哥的战旗？”
<br/>　　“嗯呐，我给他要回来了，现在送给你。”程大雷拍拍林少羽的肩膀：“一路小心。”
<br/>　　临别之际，程大雷心里有太多话要嘱托，可这嘱托究竟有用没用，程大雷也说不好。当然，最好的法子是将林少羽留在蛤蟆寨，可程大雷不想留他，确实也留不下他。
<br/>　　程大雷如此，林少羽自然也相同，手捧着盔甲，他心肠当真软了几分。可一想到杀父之仇，去意又坚决起来。
<br/>　　面对程大雷，他噗通一声跪倒，重重磕了一个头。
<br/>　　“好啦，起来吧，蛤蟆寨不兴这个。”程大雷挥挥手。
<br/>　　林少羽再拜。
<br/>　　“大当家，我这一去，不为争名，不为争权，一为报仇，二来是还天下一个太平。若有一日做到了，便奉大当家为天下之主。”
<br/>　　程大雷撇撇嘴，心想这孩子还真是四肢发达，头脑简单，连这样的傻话也说得出口。可话虽然不聪明，但心意是真的。
<br/>　　面对跪在面前的林少羽，程大雷没有再让他起来，口中忽然道：“少羽，现在我还有一样东西要送给你。”
<br/>　　“什么？”
<br/>　　林少羽下意识问了一声，却见程大雷双目炯炯，若有火焰。
<br/>　　“一件很重要，很重要的东西，你日后一定要好好珍惜。”
<br/>　　说着，程大雷手摁在林少羽头顶，心中默默念了两个字『使用』。
<br/>　　建设四级山寨时，系统曾奖励程大雷任意提升一人隐藏属性等级，程大雷本想等着日后自己再获得隐藏属性，用在自己身上，此刻终究送给了林少羽。
<br/>　　林少羽还不知发生了什么，可某种坚硬的东西已在他内心形成，他抬起头注视着程大雷的眼睛，隐隐悟到什么。
<br/>　　姓名：林少羽（名动一方的绝世武将）
<br/>　　技能：七叠破浪抢
<br/>　　隐藏属性：复仇者
<br/>　　终于，靠隐藏属性的提升，将林少羽送到了绝世等级，也希望凭此，他日后的路可以走得顺利些。
<br/>　　送别林少羽后，程大雷心情终究是有几分不开心的。既然入了这乱世，便是三分人力，七分名数，谁知林少羽日后会是怎样结果，程大雷能做的，似乎只剩下祝福了。
<br/>　　……
<br/>　　琴川关。
<br/>　　回来后，莫鸣米便病倒了，请大夫看过后，用参汤续着命，将养几日，虽然能下地走动，但整个人却瘦了几圈，再无以前的精气神，说话也变得有气无力起来。
<br/>　　请了多少名医，真正把身体养好，还不知需要多长时间。
<br/>　　只有严笛明白，莫鸣米这病不在身体，而在心里。
<br/>　　他只是无法接受这次的惨败。
<br/>　　初上战场的小兵，见过流血和牺牲，被砍掉的脑袋，流出的肠子和肾脏，血淋淋的大腿和飞过头顶的手臂……见过这些而没有疯的人，才能成为真正的战士。而能够成为一方枭雄的人，必然是见惯了尸体和鲜血，心脏早已被锻炼得无比坚硬。
<br/>　　莫鸣米就属于这种人。
<br/>　　但是，这种人并不是不脆弱，相反，他们的盔甲越坚硬，内里可能就越脆弱。
<br/>　　很显然，这一次的惨败击破了莫鸣米的生存支柱，很有可能他将一蹶不振，再也无法站起来。
<br/>　　还好，病愈之后，莫鸣米就开始处理琴川关的事情。如今琴川关的总兵力，大概有一万余人，经过这一战，实力至少损耗一半。
<br/>　　越看这些，莫鸣米就越心疼，脸上青一阵红一阵。
<br/>　　“将军，您爱惜身体。”
<br/>　　严笛递过来一杯热茶。
<br/>　　莫鸣米点点头：“那边要的货怎样了？”
<br/>　　“正在准备，怕是要拖延一些日子。”
<br/>　　“还是赶紧准备，那边可是不能得罪。”
<br/>　　“将军，不若向那边借些人来？让他们铲除程大雷。”
<br/>　　“我也有这个打算。”莫鸣米叹口气：“可是我们如果连这点小事都处理不了，那边还会重用我们嘛。唉……实在不行，就只能向那边借些兵了。”
<br/>　　二人商议到很晚，严笛才退下。他心中也明白，经历这次失败，琴川关元气大伤，但再给莫鸣米一些日子，未必不可以恢复元气。
<br/>　　只是如何解决程大雷，才是最重要的问题。难道日后，真要让莫鸣米和程大雷在琴川一地平起平坐。
<br/>　　除非，能向那边借得兵来。
<br/>　　严笛一路往回走，一路思考着这些事情。严笛单身独居，院子里只有两个书童伺候。
<br/>　　平常是有些冷清的，但严笛这人耐得住寂寞，倒也不在乎冷清。
<br/>　　今天的院子里格外安静些，书童也没有出来迎接，严笛觉得有些奇怪，撩开门帘踏入屋子。
<br/>　　“严军师，回来啦？”
<br/>　　严笛一激灵，看到程大雷坐在茶桌后，抿着一口香茶。
<br/>　　“工作真够忙的，我等你可很久了。”
<br/>(本章完)
