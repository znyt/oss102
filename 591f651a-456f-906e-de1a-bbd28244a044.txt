　　第179章
<br/>　　嘟，三级山寨达成。
<br/>　　嘟，山寨所有技能升1（齐心、治愈）
<br/>　　嘟，奖励山寨技能1（固防：城墙修复能力提升）
<br/>　　嘟，弓兵箭术提升。
<br/>　　嘟，铁匠铺生产优秀武器概率提升。
<br/>　　嘟，点将台升级，每点将一次消耗恐惧值十万，每十次点将必获得顶级人物1，每一千万有几率兑换绝世武将1 。
<br/>　　嘟，山寨拥有者等级提升1 。
<br/>　　嘟，系统商店为你开启……
<br/>　　澎拜的力量在体内肆虐，似乎有人掰开程大雷的嘴，往他嘴里塞了一颗九转大还丹，也好像有绝世高手，给程大雷灌顶传输内力：
<br/>　　『正义的少年喔，我看你一身正义，根骨非凡，必是百年难得一遇的习武天才，我这一身绝学就传给你了』
<br/>　　随之，程大雷神功天成。
<br/>　　身体内仿佛有用不完的力气，许多莫名奥妙的战场杀招出现在脑海中，对于战斗的理解瞬间提升一个等级。更重要的提升在气力上，顶级高手秦蛮双臂至少有五百斤力气，程大雷现在也差不了多少。
<br/>　　这才是升级的快感呐！
<br/>　　终于抬头挺胸重新做人，不用在像以前畏首畏尾。程大雷心情澎湃，忍不住握紧拳头大喊几声『欧耶』。
<br/>　　“大当家，没事你喊我的化名做什么？”徐神机走过来：“是有事要与我商量。”
<br/>　　“军师，请你去城头上。”程大雷做了个请的姿势。
<br/>　　“是想我去检查城防嘛，现在大家正修补城墙呢。”
<br/>　　“不，我只是希望你离我远一些。”
<br/>　　程大雷现在心情正好，不想放这么大颗毒奶在身边，更重要的是，看到他影响心情呐。
<br/>　　程大雷检查山寨和自己的信息。
<br/>　　山寨：蛤蟆寨（坚固的三级山寨）
<br/>　　拥有者：程大雷
<br/>　　人口：4531
<br/>　　技能：治愈、齐心、固防
<br/>　　恐惧值：356607
<br/>　　……
<br/>　　姓名：程大雷（名动一方的顶级山贼）
<br/>　　年龄：20
<br/>　　技能：魔王三板斧、阿飞快剑、雄辩
<br/>　　隐藏属性：一个好人
<br/>　　“大当家，大当家……”
<br/>　　徐神机此时颠颠跑回来，程大雷挠挠头：“又怎么了？不是让你去看着修城墙么？”
<br/>　　“修好了，这么快？”程大雷吃惊道。
<br/>　　“是啊，我去看了，修得又快又好，好像比新建的还结实，不信的话，大当家你可以去看看。”
<br/>　　“不必了。”程大雷想了想道。他明白这是山寨的『固防』技能发挥作用。看来，这固防等于快速恢复耐久值。就像游戏中，水晶打呀打的快被人打破了，一个固防丢过去，耐久值瞬间回满。
<br/>　　可以想象当这一幕发生对方心态会有多崩。
<br/>　　总的来说，升级到三级山寨后，山寨的实力瞬间提升一个量级，比升到二级时强上许多。
<br/>　　其实这也不难理解，就像年薪从一万提升到十万，看似不少，其实没有太大改变，充其量从贫穷提升到普通，改善下伙食什么。但从十万提升到百万，同样翻了十倍，那相差就大了。
<br/>　　轻点了一下收获，程大雷找个没人的地方屏声静息，最重要的是避开徐神机。
<br/>　　现在，还有最后一项，也是最重要的一项自己还没有了解。
<br/>　　系统商店。
<br/>　　废话，没有商店的系统能叫做系统嘛，这要传出去是会被别人笑话的，鸡怎么说，鸭怎么说，其他系统怎么说……
<br/>　　没想到，直到自己把山寨升到三级，系统商店才为自己开启。
<br/>　　程大雷开启脑海中的系统界面，发现山寨建筑除了点将台，还多了一座系统商店。
<br/>　　嘟，收取恐惧值一万。
<br/>　　他打开系统商店时，收到这样一条提示。程大雷张大嘴巴，看嘴形是一句三字经。
<br/>　　特么的，这算什么，保护费么，每次进入商店都要收取一万恐惧值。果然是山贼系统呐，开始抢劫自己这个系统拥有者了。
<br/>　　程大雷忍着肉痛，浏览商店中的商品，目前商店内只有一件商品：火药。
<br/>　　商店也是以恐惧值做货币，一箱火药都售价是：十万恐惧值。
<br/>　　特么的！
<br/>　　程大雷憋着嘴形，又发出一句三字经，系统这明显就是打劫呀。
<br/>　　本来自己这些日子攒了不少恐惧值，有三十万了，正好这次点将台升级，程大雷还想多攒点恐惧值兑换一个绝世人物玩玩。
<br/>　　毕竟，他现在获得恐惧值还是很快的，打仗就会死人，死人就会有人怕。比如赤眉统率一万人，一人给自己贡献一点恐惧值，就是一个天文数字。
<br/>　　而根据程大雷这些日子的经验判断，单人能给自己提供的恐惧值上限是1000 。多了，再吓唬也不会给自己提供恐惧值了。
<br/>　　可现在，一箱火药就要十万恐惧值。
<br/>　　十箱火药就等于一名顶级人物。
<br/>　　在这个冷兵器时代，火药有极大作用，给程大雷一千火枪手，戎族再骑术无双也是白给。能比科技的时候，谁还比战斗意志。只有穷得揭不开锅的人才会野蛮的白刃战，有钱人都是很文明的。
<br/>　　是十箱火药重要，还是一名顶级人物重要。
<br/>　　这不是一笔容易算清楚的帐，就像问一个厨师饺子馅重要，还是饺子皮重要。
<br/>　　正确答案是都重要。
<br/>　　程大雷好不容易克制住好奇心，没舍得兑换，反正商店也跑不了，自己还是多积攒些恐惧值，等到关键时候，需要什么再兑换什么。
<br/>　　绝世武将呐，一百箱火药呐……
<br/>　　这两者都是令人激动的事情。
<br/>　　程大雷激动的心情久久不能平息，他面对戎族围城一直处于能守住，或不能守住之间。但现在山寨升到三级后……
<br/>　　守住！？
<br/>　　不要问一个暴发户要吃西红柿盖饭还是鱼香肉丝盖饭，哼，至少是木须肉盖饭。
<br/>　　程大雷现在要的也绝非是守住或守不住，而是……能不能杀光他们。
<br/>　　林冲经过医治后，从病榻上醒来。醒来第一件事，就是找程大雷。
<br/>　　看他健步如飞的样子，谁能想到他原先受了很重的伤。是该说刘悲医术高明，还是该说苦逼的人生命力都比较强呢。
<br/>　　林冲见到程大雷的第一句话就是：少羽在哪里？
<br/>　　“林教头放心，少羽正在执行一个极重要的任务，现在绝无危险。”
<br/>　　“在哪里？”
