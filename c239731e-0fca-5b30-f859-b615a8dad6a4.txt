　　“有何代价？”萧逸问道。
<br/>　　听面前这总执事的话语，似乎成为执事好处多多。只是，自己还是要问清楚。
<br/>　　“很简单。”总执事笑道，“这执事之位，你需要付出1500任务点去兑换。”
<br/>　　“1500任务点？”萧逸暗暗点头，难怪刚才那个工作人员会叫住自己。
<br/>　　在猎妖殿，无论你想得到什么好处，前提都必须是任务点。
<br/>　　“恕易某直言，我成为执事后，能得到什么？”萧逸直接问道。
<br/>　　1500任务点可不是小数目，自己辛辛苦苦三个月，横扫了数之不尽的匪窝，完成了50个四级任务，才得到这些任务点。
<br/>　　若是其他普通三级猎妖师，数年都不一定攒到1500任务点。
<br/>　　虽然现在自己暂时没什么东西需要换，但这么多任务点，可不能白给。
<br/>　　总执事摆摆手，笑道，“无妨，自然是要问清楚的。刚才我已经跟你说了，成为执事，荣耀、权利、地位，你唾手可得。”
<br/>　　“易霄，虽然你戴着面具，但如果我没有感知错的话，你今年不过16岁吧。”
<br/>　　“一旦你成为执事，便是北山郡猎妖殿史上年龄最小的执事。堪称前无古人，甚至可以载入猎妖殿史册。”
<br/>　　“你说，这是何等莫大的荣耀阿。”
<br/>　　想要成为猎妖殿执事，途径只有一个，靠自己的本事，完成一个个任务，攒够1500任务点去兑换。
<br/>　　北山郡历史上，最早成为猎妖殿执事的，是一个当时惊采绝艳的天才武者。
<br/>　　年仅18岁便成为猎妖殿执事，22岁成为总执事，30岁一身修为横行北山郡，无人能敌，堪称传奇。
<br/>　　当然，单从成为猎妖殿执事这一项，现在萧逸更早。
<br/>　　不过，萧逸却是苦笑一声。
<br/>　　他还未说什么，旁边那女子已经偷笑道，“总执事，人家易霄还未死呢，载入史册做什么。”
<br/>　　萧逸也是撇撇嘴，自己要那虚名屁用都没有。
<br/>　　总执事瞪了那女子一眼，呵斥道，“你懂什么，猎妖殿乃是全大陆最古老的势力，载入猎妖殿史册，这是何等荣耀，别人求都求不来呢。”
<br/>　　说罢，总执事又看向萧逸，笑意盈盈道，“易霄，你说是吧。”
<br/>　　萧逸不语。
<br/>　　总执事见状，继续道，“易霄啊，老夫是真心希望你能加入猎妖殿的。”
<br/>　　“执事，可是异于普通猎妖师的。”
<br/>　　“普通猎妖师，与我们猎妖殿只有任务间的关系，他们一切的行为，几乎都与我们无关。”
<br/>　　“而执事，可就是自己人了，真正加入猎妖殿势力，地位之高，远超普通猎妖师。”
<br/>　　“若是在外，只要你亮出执事令牌，任何猎妖师都需向你行礼，不得冒犯。”
<br/>　　“你说，这地位高不高？”
<br/>　　总执事一脸笑意，道，“你试想一下，若是你行走在外，一个个猎妖师都对你毕恭毕敬，尊敬有加的样子，何等有面子阿。”
<br/>　　“对于你们年轻人来说，面子是最重要的吧。”
<br/>　　萧逸不禁满头黑线，在他眼中，面前深不可测的总执事，似乎成了一个贱人，在不断贱笑。
<br/>　　所谓的荣耀、地位，竟然就是这样，他感觉自己好像被耍了。
<br/>　　面前的老头，似乎在寻他乐子玩。
<br/>　　“总执事，若没有其他事，易某告辞了。”萧逸摇摇头，转身便欲离开。
<br/>　　“诶，慢着。”总执事连忙叫住萧逸。
<br/>　　“你先别急，我还没说完呢。”总执事笑道。
<br/>　　萧逸再次苦笑道，“总执事请说。”
<br/>　　总执事继续道，“荣耀、地位，你不在乎。但，权利，你总该需要的吧。”
<br/>　　萧逸脸色淡然，他似乎已经可以预料到，面前老者又会说一些无用的屁话出来。
<br/>　　总执事忽然收起了之前的笑意，脸色变得认真。
<br/>　　“我刚才说过，成为执事，便是真正加入猎妖殿势力。凭此，你可以借用猎妖殿的各种资源。”
<br/>　　“资源？”萧逸稍微有了些兴趣。
<br/>　　“不错。”总执事正色道，“譬如执法队，你可以付出任务点，请动他们为你办任何事。”
<br/>　　执法队作为猎妖殿的直接武者力量，个个的修为都不俗，最弱的都是半步洞玄。若是队长级别，起码在洞玄九重，乃至半步破玄。
<br/>　　“又譬如直接发布任务，若你行走在外，需要帮手时，可不通过猎妖殿，直接当场发任务，让附近的猎妖师为你效命。”
<br/>　　猎妖师遍布全大陆，单单百武城分殿附近十数城，就有不下数万猎妖师。
<br/>　　无论身在何处，都必定能在附近遇到为数不少的猎妖师。
<br/>　　“另外，你如果需要兑换级别太高的物品，可是需要权限的。比如地阶武技，或者五品以上的天材地宝等等。”
<br/>　　“你若成了执事，这些权限便有了。”
<br/>　　“无论怎样，你都是不亏的吧。”总执事忽然站了起来，拍了拍萧逸的肩膀，‘慈祥’地笑道。
<br/>　　萧逸苦笑一声，从长远来看，自己日后肯定是需要兑换更高级的东西的，迟早都需要成为执事的。
<br/>　　“既然易某承蒙总执事厚爱，这执事之职，我便兑换了。”萧逸点头道。
<br/>　　“好，痛快。”总执事大笑一声，对一旁的女子道，“妙妙，去把手续办了。”
<br/>　　女子起身，要了萧逸的猎妖师令牌，刚准备离去，忽然问道，“总执事，那500点奖励，现在给你记上吗？”
<br/>　　“500点奖励？”萧逸一愣。
<br/>　　女子道，“易霄先生不知道吗？为猎妖殿新添一个执事，可是能得到500点任务点奖励的。”
<br/>　　“额。”萧逸顿时满头黑线，瞥了那总执事一眼。
<br/>　　500任务点阿，做多少个四级任务才能攒到。
<br/>　　这总执事倒好，随便忽悠几句就轻易得到了。难怪他极力忽悠自己成为执事。
<br/>　　“哈哈。”总执事贱笑道，“易霄，痛快点，以后你也会遇到这些笨蛋的，忽悠几句，这些任务点总会赚回来的。”
<br/>　　“笨蛋？”萧逸的脸完全黑了起来，现在自己就是那个大笨蛋。
<br/>　　这时，女子已经离去。
<br/>　　总执事扔给萧逸一本书，道，“易霄，别难过，你看看这物品手册，可是你成为执事后才能有权限兑换的宝贝。”
<br/>　　萧逸接过书册，随便看了眼，脸色黑得更加厉害。
<br/>　　上面的东西，随便一样都需要过万任务点，甚至过十万任务点才能兑换的东西。
<br/>　　东西是宝贝，可自己何年何月才能兑换得到阿？
<br/>　　说来说去，自己现在亏了1500任务点，啥都没得到。
<br/>　　这总执事，看着深不可测，没想到如此老奸巨猾，竟被他摆了一道。
<br/>　　好不容易攒了几个月的任务点，一下子就被骗光了。
<br/>　　萧逸无聊地翻着那物品手册，权当开开眼界了。
<br/>　　忽然，他的目光定格了一下。因为，手册上，竟有一东西只需2000任务点就能兑换，特别显眼。
<br/>　　“猎妖令？”萧逸眼睛一亮。
<br/>　　猎妖令，得此令者，其家人或家族，受猎妖殿庇护。
<br/>　　“猎妖令？”总执事停下了贱笑，问道，“怎么，易霄，你有需要保护的家人？”
