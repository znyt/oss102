　　一望无际的冰雪荒原之上，
<br/>　　一头有着九只翅膀、通体紫色的庞大妖兽，朝南方呼啸而去。
<br/>　　而离近看，在那妖兽背上，居然还有一男一女两个人。
<br/>　　那男的，也不过二十左右，一身白色衣袍，盘腿坐在妖兽背上，正在打坐修炼，周围狂风呼啸，却对他丝毫没有任何影响，就如同整个人黏在了妖兽身上一般。
<br/>　　此人自然正是叶尘！
<br/>　　而在叶尘一旁，还有着一名身穿红色衣衫的美貌女子，此时两眼之中，充满着浓浓的怨恨之色。
<br/>　　这美貌女子，自然正北寒城城主的女儿，北山薇儿。
<br/>　　叶尘之所以要挟持北山薇儿，一来是想给这女人一个教训，二来则是为了牵制脚下的这头九翅紫电鹫。
<br/>　　这九翅紫电鹫，乃是四阶妖兽，本身就极具灵性，而且飞行速度极快，即便是叶尘也远远不能及。
<br/>　　倘若没有北山薇儿这个人质，难保这畜生不会搞出什么事端来。
<br/>　　“喂，我们都已经赶了快一天的路，是不是也该休息一下了？”
<br/>　　北山薇儿见叶尘坐在那里一动不动，也不说话，忍不住鼓足勇气开口道。
<br/>　　叶尘依旧没有吭声，继续暗暗运转体内真元，修复着体内的伤势。
<br/>　　北山薇儿又一连说了几句话，见叶尘都不理她，胆子也渐渐大了起来，
<br/>　　“喂，我警告你啊，等到了天玄城，你必须把我给放了，否则我父亲和整个北寒城，都不会放过你的！”
<br/>　　叶尘依旧不予理会。
<br/>　　北山薇儿顿时气得不行，正打算狠狠痛骂叶尘一顿。
<br/>　　然而就在这时，叶尘却猛然睁开双眼，
<br/>　　“闭嘴！”
<br/>　　北山薇儿见叶尘发怒，还以为是自己之前的话，把叶尘给激怒了，顿时吓得他缩了缩脑袋，不敢再多说一句。
<br/>　　叶尘并没有多看她一眼，而是缓缓站起身来，目光望着前方，眉头微微一皱。
<br/>　　只见，数百里外，天空之上，竟然有成千上万头飞行妖兽，铺天盖地而来，几乎将天空都遮蔽住了！
<br/>　　“不好！前方出现了兽潮，我们必须要绕道避开！”
<br/>　　一道苍老的声音响起，却是那九翅紫电鹫，忽然口吐人言开口道。
<br/>　　“不行！”
<br/>　　叶尘直接一口拒绝，不容置疑道：
<br/>　　“九翅紫电鹫，你只管按照原路线往前走，其他的事情，我自会解决！”
<br/>　　一旦绕道而行，必然会浪费更多的时间，这不是叶尘愿意看到的结果。
<br/>　　九翅紫电鹫顿时怒声道：
<br/>　　“你这样会害死我们的，你知不知道？我可不想陪你去送死！”
<br/>　　九翅紫电鹫说着，便要闪动翅膀，准备改变飞行路线。
<br/>　　“哼！这可由不得你！”
<br/>　　叶尘见状，顿时冷哼了一声，手掌猛然抬起，朝九翅紫电鹫的硕大脑袋上一拍，同时一声低喝，
<br/>　　“御魂！！”
<br/>　　嗡~~
<br/>　　九翅紫电鹫顿时浑身一颤，继而感觉自己的身体，仿佛被完全控制住了一般，自己已经无法左右。
<br/>　　“你！你对我做了什么？”
<br/>　　九翅紫电鹫大急，忍不住大吼大叫了起来。
<br/>　　叶尘冷冷一笑，也懒得解释，
<br/>　　他刚刚所用的，乃是专门克制妖兽的御魂之术。
<br/>　　妖兽力量强悍，但是神识普遍偏弱，而且叶尘又是突袭之下，即便九翅紫电鹫是四阶妖兽，依旧没有丝毫抵抗之力。
<br/>　　百里距离对于他们来说，不过就是一转眼的功夫，那些漫天的飞行妖兽，已经出现在眼前，看到两人一兽，顿时兴奋的嚎叫着，朝他们飞扑了过来。
<br/>　　北山薇儿看到此情此景，顿时俏脸惨白，不由得气得吼了起来，
<br/>　　“你是疯了吗？明知道前方有兽潮，还往前冲？你会害死我们的！！”
<br/>　　叶尘一言不发，
<br/>　　眼看那些飞行妖兽，咆哮着冲了过来，叶尘猛然张口，
<br/>　　“啨！~~”
<br/>　　一道犹如龙吟般的声音，陡然自叶尘的喉咙中发出，化成了一道道音波，朝四面八方蔓延而去。
<br/>　　而那些飞行妖兽，听到这道声音，顿时犹如老鼠见到了猫，吓得纷纷四散而逃，自动让出了一条通道出来。
<br/>　　“这……”
<br/>　　北山薇儿看到这一幕，顿时用玉手捂着嘴巴，惊愕的说不出话来。
<br/>　　就连两人脚下的九翅紫电鹫，也发出了一道惊呼声，一双巨眼之中，透出浓浓的震惊之色，
<br/>　　“天龙之音！这是天龙之音！你一个人族，怎么可能会发出天龙之音？”
<br/>　　叶尘淡淡一笑，并没有过多解释。
<br/>　　他体内有神龙之血，加上前世曾经得到过一门龙族秘术，所以才能发出天龙之音。
<br/>　　龙族，乃是妖族至尊！
<br/>　　尽管叶尘的天龙之音并不纯粹，可也足吓退这些低阶妖兽。
<br/>　　“啨~~啨~~”
<br/>　　叶尘又一连发出了几道天龙之音，终于从这股兽潮中，安然无恙的穿梭了出来。
<br/>　　“我们竟然从兽潮中活下来了？”
<br/>　　北山薇儿拍了拍初具规模的胸脯，一脸的不可思议之色。
<br/>　　要知道，面对这种大规模的兽潮，即便是化神天君，也不敢轻易正面对抗。
<br/>　　叶尘带着他们，不但从兽潮中活了下来，而且毫发无损，这简直令人难以置信！
<br/>　　而那九翅紫电鹫，经过这件事之后，对叶尘也变得无比驯服起来。
<br/>　　然而，
<br/>　　还没等他们两人一兽松一口气，前方一处山谷之中，陡然发出一道咆哮之声！
<br/>　　继而，
<br/>　　“吼~~”
<br/>　　一道庞大的身影，自那山谷之中飞出，直奔他们呼啸而来。
<br/>　　这竟是一头巨蟒，身躯长达近百米，周身全身黑色的鳞片，而且腹下竟然还有生两对硕大的肉翼！
<br/>　　“这是……五阶妖兽，吞天蟒！！原来之前的兽潮，便是它引发的！”
<br/>　　九翅紫电鹫忽然大呼了一声，同时在空中绕了一个弯，险而又险躲开了那头巨蟒的攻击。
<br/>　　“五阶妖兽？”
<br/>　　叶尘听到这话，也不由得眉头微微一皱，
<br/>　　五阶妖兽和化神境相当，而且气血浑厚，即便是他底牌尽出，想要诛杀五阶妖兽，也不是一件容易的事情。
<br/>　　“逃！”
<br/>　　叶尘当机立断，对脚下的九翅紫电鹫下达命令道。
<br/>　　九翅紫电鹫听到叶尘这个命令，自然是求之不得，哪里还敢犹豫，九只翅膀快速闪动，当即带着两人狂奔。
<br/>　　而那吞天蟒却丝毫没有要放弃的打算，紧跟其后追了上来。
