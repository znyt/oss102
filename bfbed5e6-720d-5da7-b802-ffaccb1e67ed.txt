　　第594章
<br/>　　程大雷坐镇蛤蟆寨时，一直派出人手收集各方面情报。他当然没忽略戎族，只是在这方面收效不大。
<br/>　　关键是戎族不成体制，数百个部落分布在草原上，今天这个部落将那个部落灭了，过两天又冒出几个小部落，情况极其复杂。
<br/>　　草上飞的日子过得很糟糕，戎族太大的部落他不敢抢，太小的部落也委实没什么好抢的。他主要是靠抢劫过往的行商过活，只不过最近草原上没什么商队。
<br/>　　见到程大雷后，他丝毫没要反抗的心思，只盼望着将这尊瘟神赶紧送走。他将自己了解到的事情一五一十告诉程大雷，确实也没有什么又用的信息。而燕不归的名字，他也并没有听说过。
<br/>　　程大雷听罢之后有些失望，本指望从草上飞这里得到些有用情报，结果仍旧是一无所获。
<br/>　　“程当家要打听消息，为什么不去戎族王城问问，那里人毕竟多，想打听谁也容易。”草上飞心里巴望着程大雷早点走。
<br/>　　关于这一点，程大雷也不是没有想过。不过他和小白狼两个人，怎么改扮都不像戎族，进到戎族窝子，那不是报仇，是直接去送死。
<br/>　　到此时，程大雷总算明白这件事的难处。在茫茫草原上，想找到一个人谈何容易，而他也没有孤身进入戎族王城的勇气。
<br/>　　看着程大雷闭口不说话，草上飞试探的问道：“程当家已决定了，是要去王城探探？”
<br/>　　程大雷目前还没疯，他摆手叹口气道：“戎族的王城我就不去了，你也知道，他们不怎么欢迎我，我就在你这里待几天。”
<br/>　　“待几天……”草上飞立刻睁大眼睛，满脸苦涩。
<br/>　　“怎么，你不开心？”
<br/>　　“开心，开心。程当家能待在我这里，实在是我八辈子积德，祖上修来的福分。”
<br/>　　“也别太高兴，这几天你帮我在附近打听打听，吃住我不怎么挑剔，过得去就行。”程大雷随口道：“下去准备吧。”
<br/>　　草上飞强挤出笑容，比哭还难看，等走出屋子，他一张脸立刻耷拉下来。嘴里骂骂咧咧，心中十万八千个不痛快。自己不知倒了哪路霉，竟然招惹到这等煞星。如今，草上飞只能自己安慰自己，毕竟，他至少没有一剑把自己宰了。
<br/>　　程大雷倒不是看中了草上飞这个地方，委实是这一路从南到北，风餐露宿，程大雷受了不少颠沛之苦。也就是想借草上飞这个地方养养精神，慢慢收集草原上的情报。
<br/>　　“大哥，咱们真容他在这里嚣张跋扈？干脆弟兄们一起动手，做了他得了。”草上飞刚走出屋子，便被手下人迎住了。
<br/>　　“呔，住口。”草上飞吓了一跳，四周看看，压低声音道：“这话不要说了，和他动手，嫌命长了么。咱们再多几倍人，也未必是他的动手。”
<br/>　　“难道真就这样容着他，这口气，我实在咽不下去。”
<br/>　　“咽不下去也得咽，你以为我就忍得了么，这不是没办法的事么。”
<br/>　　草上飞垂头丧气，心中八百个不乐意，也得对程大雷笑脸相迎。他对程大雷好吃好喝好伺候，一面派出人手，去为程大雷打探情报。
<br/>　　草上飞手下不过五六十人，不过抢劫一般的商队也足够用了。这次草上飞亲自行动，想着能打听到一些有用的消息，趁早将程大雷这尊瘟神送走。
<br/>　　可茫茫草原，除了豺狗，野狼，连个人影都遇不见，更不用提打探什么消息了。
<br/>　　黄昏时分，草上飞在一棵大树下歇脚，一边啃着干粮，嘴里一边骂骂咧咧。
<br/>　　“老大，这样的日子什么时候是个头，弟兄们实在受够了。”
<br/>　　“你们难受，难道我就不难受么，可又有什么法子。”
<br/>　　正说着，草上飞看到一队人骑着高头大马从面前经过，人人锦衣华服，虽然风尘仆仆，但也能看得出来是大户人家出来的。
<br/>　　草上飞倒是很想劫了对方，但看到对方身上携带的铁剑时，便打消了这个主意。
<br/>　　“现在这日子熬一天是一天吧，等什么时候那程大雷走了，咱们就熬出头了。”草上飞发出一声长叹。
<br/>　　本来一行人已经从他们身边经过，可突然又掉过头来，为首之人凶神恶煞道：“你们刚才说的是谁？”
<br/>　　“程大雷啊，怎么了？”草上飞吓了一跳，随之反应过来，立刻道：“你们是什么人，问老子这个干嘛？”
<br/>　　一行人对视一眼，都从彼此眼神中看出了欣喜。他们一行人，正是从琅琊城千里迢迢追过来的钟牧山一伙。
<br/>　　在进入草原后，他们遇到和程大雷一样的麻烦，草原太大了，想要寻到一个人谈何容易。不过，他们比程大雷的运气好一些，无意中竟碰到草上飞一伙。
<br/>　　钟牧山问道：“程大雷在哪里，快点儿告诉我？”
<br/>　　“你们和程大雷有仇？”
<br/>　　草上飞眼前一亮，这伙人莫非和程大雷有仇。可他转念一想，凭这几个人也未必是程大雷的对手。别他们杀不了程大雷，反倒把自己折进去。
<br/>　　现在的情况的确很糟糕，但不代表不能更糟糕。
<br/>　　“我自然知道程当家在哪里，但我绝不会告诉你们。”草上飞冷哼一声，显得十分硬气。
<br/>　　如果按照钟牧山原先的脾气，当时就要拔剑。不过，这一路走过来，他们也吃了许多苦头，本身已经不是初出茅庐的雏儿。
<br/>　　钟牧山摁住蠢蠢欲动的同伴，向草上飞做了个揖，然后竟带着人离开了。
<br/>　　草上飞看着他们远去的背影，大喝一声道：“走，我们回去。”
<br/>　　一行人踏上返回山寨的路，手下缀在草上飞马后，道：“老大，他们跟在后面呢，要不把他们甩了？”
<br/>　　既然敢号称马匪，本身骑术还是不错的。不过草上飞摆摆手，道：“不用，就当没看见，让他们跟着。”
<br/>　　一路回到了老巢，草上飞先去见了程大雷。程大雷搬了张椅子坐在外面晒太阳，见草上飞回来，问道：“怎么样，今天有没有打探到什么有用的消息？”
<br/>(本章完)
