　　第518章
<br/>　　程大雷正在房中呼呼大睡，他自然知道白衣老尼不会善罢甘休，但人总要吃饭睡觉，不能因为别人要杀自己，便不过日子了。
<br/>　　此刻他睡得正着，冷不防外面响起『哞哞』地声响，程大雷这点警觉还是有的，腾地从床上坐起，眼睛睁得铜铃一般大。
<br/>　　“怎么了？”同床共枕的樊梨花被他吵醒。
<br/>　　程大雷也不知道怎么会事，但这叫声一声比一声高，一声赛一声急。他听得心慌意乱，披上袍子从床上下来。
<br/>　　“我出去看看，会不会是有人要偷我的牛。”
<br/>　　程大雷推门而出，大踏步向后院马圈走去。这墨玉乌月牛，是程大雷的心爱之物，绝世坐骑，哪里能寻得第二头。许多场战斗，还多亏这头黑牛出力，程大雷是真破它有什么闪失。
<br/>　　樊梨花迷迷糊糊被吵醒，便再难入睡。关键那哞哞的声音不停响在耳边，她也没办法睡着。索性就陪着程大雷出去看看。女人穿衣服自然比男人麻烦许多，过了好大一会儿，她才披衣下床，沿着程大雷走过的路，同样走向后院。
<br/>　　程大雷出得屋来，脚尖一点地，便跃上了屋脊。沿着房顶往前走，没有多久便来到后院。
<br/>　　待看到后院的情景时，顿时吓了程大雷一跳。只见一个白色人影，手中捧着一黑漆漆的物事，另一只手正在挪井上的石头。
<br/>　　若非程大雷也算见过一些世面，还以为这白衣人影是从井中爬出来的鬼物。定了定神，才算认出这人是想杀自己的白衣老尼。
<br/>　　“呔，你要做什么！”
<br/>　　程大雷站在房顶上大吼一声，喊完他就后悔了。他出来的急，也没带兵器，对付这样的高手，可是不敢有半分托大。赤手空拳与对方过招，死的很有可能是自己。
<br/>　　白衣老尼也是一激灵，毕竟做贼心虚，正在做坏事时被人逮个正着，她不能不害怕。关键是陶罐中的蛤蟆还一声一声叫着，它安静了一路，此刻却不知怎地兴奋起来。
<br/>　　暂时停下手中的动作，怒目盯着程大雷，心中也有些犯怯。上次在程大雷手中就没讨到好处，这次自己是深入敌营，但凡有个闪失，很可能自己就出不了这座蛤蟆城。
<br/>　　二人一个房顶，一个院中，互相都板着脸，显得杀威凛凛。但其实，心里都有些打鼓。尤其是程大雷，最关键是他不知道白衣老尼要干什么，就见她手中托着一黑漆漆的物事，还不停发出古怪叫声，实在不知道是什么东西。
<br/>　　任程大雷想破脑袋，也想不明白白衣老尼的目的。
<br/>　　“啊呀！”他忽然怪叫一声，指着白衣老尼破口大骂：“你是要用妖术害我么！”
<br/>　　白衣老尼手中的东西，实在像极了某种邪恶的法器。莫非是招魂铃，丧命鼓之类的东西，只要一将这东西打开，听到声音的人便魂飞魄散。
<br/>　　不得不说，程大雷的想象力有些忒惊人了。
<br/>　　二人的对峙并没有持续太久，白衣老尼突然托起井上的石头，口中大叫一声，将石头抛了过来。
<br/>　　“狗贼，去死！”
<br/>　　当真好大的力气！
<br/>　　程大雷心中暗暗赞了一声，单手就能将井口的石头抛上房顶，这力气无论如何也不简单。不过，若是比拼力气的话，程大雷不惧任何人。这白衣老尼毕竟是个女人，女人相比男人，在力气上天生不占优势。
<br/>　　程大雷口中喝了一声，双手撑起，硬生生接住了抛过来的石头。然后身体发力，重新砸向白衣老尼。
<br/>　　老尼姑轻巧避开，一手抱着陶罐，另外一手的拂尘已经抖开。这个时候，程大雷也从屋顶上跃下了，拳脚功夫，程大雷本身也不擅长，赤手空拳他几乎不知道该怎么打。
<br/>　　不过，他在跳下来时，已经看到草堆中一木叉，这是平时翻草料用的。程大雷跃下来便握住了木叉，抖了抖，在身前甩开，将逼过来的老尼姑击退。
<br/>　　程大雷手中握了兵器，豪横气顿生，自己还对付不了一个老女人么。他抖索精神，逼近老尼姑，手中的木叉狠狠甩下来。
<br/>　　这是刑天斧的打法，一击不中，他又刺了一招，逼的老尼姑节节败退。
<br/>　　老尼姑也知道今天的情况不善，但逼到绝境，反而激起了她的战意。手中拂尘缠在木叉上，力从地起，只听咔嚓咔嚓声响，生生将木叉扭断。
<br/>　　程大雷心中一怯，刚才的木叉变成一截木棍。毕竟是兵器不占优势，手中的木棍可没有匹夫剑用得顺手。
<br/>　　白衣老尼一看占了上风，那还可能放过程大雷，越逼越急，就要这里杀死程大雷。程大雷只能用轻身功夫躲避，越来越狼狈，稍不留神，老尼姑的拂尘就会在身上砸出个血窟窿。
<br/>　　“呔，接着！”
<br/>　　樊梨花的声音响起，匹夫剑抛了过来。她出门比程大雷慢了半拍，临出门前将程大雷的匹夫剑捎上了。此刻来到门口，见到这一幕，忙将匹夫剑抛向程大雷。
<br/>　　程大雷抄手接住，有了趁手兵器，实力陡然大增，只用一招，就逼退了白衣老尼，逆转了战斗的形势。
<br/>　　白衣老尼不敢再强攻，她旧伤未愈，速度和反应不能和全盛时相比。此刻程大雷手中又握住了剑，再斗下去，怕自己不是她的对手。
<br/>　　此时，她注意到了门口的樊梨花，杀程大雷有些够呛，杀这樊梨花岂非手到擒来。当真是有些走火入魔，身子陡然扑向樊梨花。
<br/>　　程大雷飞身赶到，一剑拦住了白衣老尼，二人厮杀斗在一起。
<br/>　　“退！”程大雷冲樊梨花大吼一声。
<br/>　　樊梨花也知道自己帮不上什么忙，立刻退出了院门，把战场留给程大雷和老尼姑两人。
<br/>　　二人打得难分难解，但终究是程大雷占了上风。论境界，论战斗经验程大雷都比不过老尼姑，可是，他毕竟年轻，从来拳怕少壮，乱拳就可以打死老师父。
<br/>　　这时候哞地一声，白衣尼姑将手中的陶罐抛向程大雷。程大雷不知道这是什么古怪物事，用单手托住，轻飘飘搁在脚底下。
<br/>(本章完)
