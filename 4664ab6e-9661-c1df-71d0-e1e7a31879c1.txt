　　听到这话，叶尘顿时两眼一瞪，当即道：
<br/>　　“那贵族现在在何处？你立即带我去找他！”
<br/>　　既然找到了玫瑰的线索，叶尘自然一刻也不想耽搁。
<br/>　　“哎呦~~”
<br/>　　叶尘拉着那魔族美女，就打算离开，不料这女人竟然身体一软，倒在了他的身上，而且还用那饱满的部位，在他身上蹭了蹭，娇声道：
<br/>　　“小弟弟，你好粗鲁啊，把姐姐都弄疼了~”
<br/>　　罗刹女天生淫荡，果然不假！
<br/>　　而且这女人，很显然也料定了叶尘，一时半会不会杀她，所以才敢这么放肆。
<br/>　　“哼！”
<br/>　　叶尘冷哼一声，松开魔族美女的手腕，然后手掌一翻。
<br/>　　呼！
<br/>　　三昧真火已经重新出现在他的掌心之中。
<br/>　　“去！”
<br/>　　叶尘屈指一弹，三昧真火瞬间便射入魔族美女的体内！
<br/>　　“你你……你对我做了什么？”
<br/>　　魔族美女顿时脸色狂变，瞬间变得惨白无比。
<br/>　　她刚刚可是亲眼看到，叶尘就是用这火焰，将他的那名手下，烧成了一缕青烟。
<br/>　　“放心！一时半会你还死不了！不过接下来，你必须要听命于我，只要你乖乖听话，我自然不会杀你！否则的话……”
<br/>　　说到此处，叶尘的身上，升起一股浓浓的杀意！
<br/>　　魔族美女顿时吓得浑身一颤，再也不敢放肆，当即乖乖低头道：
<br/>　　“是！”
<br/>　　……
<br/>　　通过叶尘的一番询问，很快便对整个魔界，有了一个大致的了解。
<br/>　　整个魔界并不大，按照势力来划分的话，一共分为九大势力，分别对应的是九大主城。
<br/>　　他目前所在的罗刹城，是九大主城之中，势力最弱的一方，而最强的则是魔界的中心，魔皇城！
<br/>　　至于这位魔族美女，名叫戴安娜，是城主府四大侍卫长之一，实力为凝血境，相当于修仙者中的筑基境。
<br/>　　当然，虽然她的境界和叶尘旗鼓相当，不过真实实力，可就相差太多太多了。
<br/>　　“叶先生，我们必须要先回一趟城主府，调查清楚那位贵族的行踪，然后才能找到您要的东西！”
<br/>　　戴安娜小心翼翼的开口道。
<br/>　　虽然戴安娜实在想不明白，眼前这个人族男子，怎么会有如此强大的实力，可事实摆在眼前，而且自己的性命，如今也被对方所控制，她也只能老老实实听命。
<br/>　　叶尘点了点头，
<br/>　　“那就走吧！”
<br/>　　不料，戴安娜却又支支吾吾道：
<br/>　　“叶先生，那个，寒灵林是我罗刹族的禁地，周围有重兵把守，您这个样子恐怕……”
<br/>　　叶尘轻轻一笑，
<br/>　　“这个简单！”
<br/>　　说完这话，叶尘心念一动，整个人瞬间拔高了一截，变成了一名两米开外的壮汉，而且气息也随之改变，周身魔气缭绕。
<br/>　　一旁的戴安娜，看到叶尘如同变了一个人一般，顿时眼珠子都快要瞪出来了，心中对叶尘的手段，越发感到震撼。
<br/>　　……
<br/>　　很快，
<br/>　　叶尘穿上戴安娜为他找了的一件侍卫盔甲，冒充成她的手下，离开这片树林。
<br/>　　出了树林之后，往前走了近百里，前方出现一座十分雄伟壮阔的城池，应该就是戴安娜口中的罗刹城了。
<br/>　　魔界之中，灵气浓郁，资源丰富，生存环境比之前的妖界，还要好上不少。
<br/>　　这让叶尘多少有几分，重回修真界的感觉。
<br/>　　不一会，
<br/>　　叶尘跟着戴安娜，来到城池之前，发现在城门口，有着一大片空地。
<br/>　　而在那片空地之上，竟然是一排排铁笼子，至于铁笼子里面装的，则全都是人族，这些人个个目光呆滞，蹲在铁笼子之中，犹如待宰的羔羊一般。
<br/>　　而在那些铁笼子周围，则是一群群身材高大的魔族之人，冲着那些铁笼子指指点点。
<br/>　　“叶先生，那里是奴隶拍卖场……”
<br/>　　戴安娜也留意到了叶尘，当即小心翼翼的解释了一句，但是又不敢多说，生怕惹得叶尘发怒，做出什么出格的事情来。
<br/>　　毕竟，在魔界之中，人族是最低等的奴隶，甚至是魔族的食物，这个已经是公认的事情了。
<br/>　　叶尘点了点头，并没有多少什么，
<br/>　　前世在修真界，他也曾经踏入过魔族腹地，所见识过的场景，比眼前这些，还要残忍千百倍！
<br/>　　他倒也不是已经麻木，只是现在要尽快找到玫瑰的肉身，其他的事情，没有功夫去理会。
<br/>　　戴安娜见叶尘神色如常，顿时松了一口长气，连忙带着叶尘朝城门口走去。
<br/>　　城门巍峨壮观，宽阔高大，足足有数十米高。
<br/>　　戴安娜身为城主府的四大侍卫长，在罗刹城也属于贵族行列，门口的守卫自然不敢阻拦。
<br/>　　然而，
<br/>　　就在两人准备进入城中之时，
<br/>　　后方忽然传来一道呼啸声！
<br/>　　“前面的人！速速让开！”
<br/>　　叶尘回头一看，只见一群魔族之人，骑着一头头似牛非牛、似马非马的庞大怪兽，从远处奔腾而来。
<br/>　　叶尘见状，不由得眉头微微一挑，
<br/>　　“灵兽？”
<br/>　　原来这些魔族之人所骑的，乃是修真界极为常见的低级灵兽。
<br/>　　不过自从叶尘重生以来，这还是他第一次见到！
<br/>　　那些灵兽的速度极快，转眼之间就已经来到城门口，而且因为这些灵兽的体积庞大，数十米宽的城门，竟然都有些拥挤。
<br/>　　“叶先生，这是十七公主亲卫队，十七公主是城主大人最疼爱的女儿，我们还是先避让一二吧！”
<br/>　　这时一旁的戴安娜，忽然冲着叶尘开口道。
<br/>　　叶尘眉头微微一皱，他可没有给别人让路的习惯，不过眼下还有要事等着办，自然没有功夫浪费这个上面，于是随着戴安娜，缓缓走到一旁路边。
<br/>　　而与此同时，那群骑着灵兽的魔族众人，也已经呼啸而至，从两人身旁擦肩而过。
<br/>　　然而，
<br/>　　让叶尘万万没有想到的是，
<br/>　　其中一名魔族青年，从叶尘身旁奔驰而过的时候，忽然扬起手中的长鞭，便朝他狠狠抽了过来，
<br/>　　“不长眼的东西！磨磨蹭蹭什么！”
<br/>　　啪！
<br/>　　一道清脆的鞭声响起，犹如一道黑色闪电般，朝叶尘狠狠劈落！
<br/>　　叶尘顿时眉头一皱，直接随手一抓，便将那长鞭一把擒住，
<br/>　　“嗯？”
<br/>　　那魔族青年显然没有想到，一个穿着普通侍卫盔甲的下人，竟然可以轻易抓住了他的鞭子！
<br/>　　“找死！”
<br/>　　魔族青年大喝一声，猛然一拉，
<br/>　　却不料，对方犹如钉在了地面之上，竟然纹丝不动！
<br/>　　反倒是他，连人带兽，滚落在了地上，连带着后面的人，也全都人仰马翻。
