<br/>    龙藏之地。
<br/>    
<br/>        陈明脚踏祥云在一座山峰上落下，同时从祥云上下来的，还有紫霞仙子，刘玄刹，雪寻梅，苏青罡。
<br/>    
<br/>        四位大君刚落下祥云，只觉得天旋地转，两眼发昏，腹中食物一阵翻涌。
<br/>    
<br/>        远处在龙藏之地外准备进入其中探索的诸王，都是看向山峰之上，“快看，那是大君到了！”
<br/>    
<br/>        “他们是准备搏杀这龙藏之地之中的恶蛟吗？”
<br/>    
<br/>        “除去这恶蛟，我们就可以进入其中寻宝了！”
<br/>    
<br/>        “为啥几位大君都在左摇右晃，是在施展什么神秘的术法吗？”
<br/>    
<br/>        “一定是了，大君正在施展秘术！”
<br/>    
<br/>        刘玄刹双手揉头，再次看向陈明脚下云彩上的那两句话，天高地也阔，安全第一条，腾云不规范，师徒两行泪。
<br/>    
<br/>        这两句话真的是认真的吗？这tm叫腾云，这叫腾雷还差不多！“陈明，我再乘你的云，我就不是人！”
<br/>    
<br/>        紫霞仙子捂额，面色憔悴，“我觉得你可以把这两句话扣了，我真是信了你的邪，才上了你的云。”
<br/>    
<br/>        陈明大摇大摆道:“不是没事吗？说的好像你们谁受伤了一样。”
<br/>    
<br/>        紫霞仙子有气无力道:“内伤！”
<br/>    
<br/>        紫霞仙子忽然道:“对了，据说你们燕山已经介入王级战场，真的假的啊，老君们让你们燕山撤出战场，没事不要去瞎添乱，炼丹师，好好炼丹不就完了。”
<br/>    
<br/>        陈明道:“我现在又不在燕山，燕山发生了什么事，我怎么知道，估摸着是卓青瑶那个小妮子手痒了呗！”
<br/>    
<br/>        紫霞仙子忽然面色一变，变得有些诡异，然后看向陈明，“上次你们攻打妖族大君不是燕山的全部兵马吗？”
<br/>    
<br/>        陈明道:“你怎么这么问啊？”
<br/>    
<br/>        紫霞仙子道:“在我们腹地的战场之中，至少发现了五六十万的赤潮大军，赤潮大军攻城拔寨，速度奇快无比，目前已经将战场推进一千里了。”
<br/>    
<br/>        陈明呵呵一笑，道:“这我就不知道了，管他的呢，就让燕山的弟子前去锻炼一二吧。这龙藏之地，到底有着什么玄机？”
<br/>    
<br/>        紫霞仙子道:“传闻在龙藏之地中，有着一条道龙”
<br/>    
<br/>        陈明打断了紫霞仙子，问道:“等等，金龙，青龙，火龙什么的我知道，这个道龙是什么鬼啊？”
<br/>    
<br/>        雪寻梅解释道:“就是一条鳞片上刻满了经文的龙，传闻整个大陆之上的所有经文都是出自这一条道龙，大陆上的第一位修行者，同样是第一位仙，蟠龙大仙，便是观此龙悟道，开创了修行之路。”
<br/>    
<br/>        陈明的眼中一亮，问道:“也就是说，这条道龙的身上有着很多的经文？”
<br/>    
<br/>        雪寻梅道:“确实是这样。”
<br/>    
<br/>        陈明再次问道:“那么此地又是为何成为了一处禁地的？按理来说，此处应该是一处福地才对啊。”
<br/>    
<br/>        雪寻梅道:“蟠龙大仙飞升之后，众多的修行者对道龙起了贪欲，没有了蟠龙大仙的镇压，贪欲也压抑不住了，修行者们想要屠了这一条道龙，获取它身上的经文。故而蟠龙盛怒，此后见到修行者便屠杀殆尽。”
<br/>    
<br/>        陈明道:“听这道龙这么厉害，我们几个，真的能够拿下它？”
<br/>    
<br/>        刘玄刹道:“道龙已经到了老年了，即便是龙，寿命也是有尽头的，更何况道龙实际上并不算是龙，至于它到底是什么，谁也不知道，老年的道龙，早已不复当年之威了。”
<br/>    
<br/>        陈明扫了一眼系统小姐姐发布的，关于龙藏之地的任务，奖励是未知的，未知的一般都有着好东西，陈明也是升起了浓厚的兴趣，每一片鳞片，都是一种经文，这简直天助我也！
<br/>    
<br/>        现在的师德，修行个百来种经文到道种境不在话下，法力可以翻一倍！
<br/>    
<br/>        哪还有什么好说的，削他！
<br/>    
<br/>        几人朝着龙藏之地的深处走去，潮湿的密林并不能阻挡他们的前进，至于原因，很简单，都是用飞的了，还怕潮湿的密林，开什么玩笑！
<br/>    
<br/>        飞了上千里，陈明的眼中，忽然看到一个浑身鳞片的异人，在密林之中一闪不见，陈明手中万道天竹一扫，数千柄灵剑飞出，直接将那个浑身鳞片的异人逼得无路可走，几人在那个异人前落下，看向眼前的异人，“这是什么？”
<br/>    
<br/>        刘玄刹道:“这应该是龙伥，那一条道龙的爪牙。也就是那一条道龙的狂信者。”
<br/>    
<br/>        异人看向五人，没有半点畏惧之色，“你们又来了吗？欺师灭祖的败类？”
<br/>    
<br/>        刘玄刹并没有露出任何尴尬的神色，“如果说道龙真的是这一块大陆的万道之源，那么它不会让你们变成如今的这个样子。”
<br/>    
<br/>        异人口中狂笑着，“哈哈哈，说的好听，道龙赐予你们大道，你们就是这么回报它的吗？生于远龙之地，长眠于龙身之下，这是我的荣耀，不要以为你们的行动很隐秘，你们已经被发现了，龙君正在赶来，你们终将一死以谢罪天下！”
<br/>    
<br/>        陈明看向这个异人的双眼，有的只是疯狂，“看来这道龙，也不是什么好东西。”
<br/>    
<br/>        不过片刻，陈明看得到在丛林之中，一位额头生着两只玉角的年轻人来到此处，这么近的距离，他们竟然没有发现，来人的修为，绝对深不可测。
<br/>    
<br/>        紫霞仙子摸出了仙器，弯弓搭箭，刘玄刹伸手取下了背后的黑色镰刀，苏青罡拔剑，雪寻梅道袍一鼓。
<br/>    
<br/>        陈明看向这修为深不可测的年轻人，眼中所见，只是问号，唯一可以看得到的，就是危险级别:陆沉。
<br/>    
<br/>        陆沉是个什么概念，就是说足以把这座大陆打沉吗？
<br/>    
<br/>        这危险指数，绝对是破了五千的存在，这要怎么打？
<br/>    
<br/>        陈明可是知道，危险指数越高，每一点危险指数的差距就越大，危险指数破了五千的存在，至少要他们五个都在三千之上才有的打。
<br/>    
<br/>        之前的那个异人，看到年轻人的时候也是眼中露出一抹喜色，“龙君大人，快快斩杀这些忘恩负义之辈，护佑祖龙！”
<br/>    
<br/>        龙君缓缓朝着五人走来，五人只感觉浑身汗毛竖起，一阵恐怖的杀意将所有人笼罩，龙君抬手，金色的龙爪已经呼啸而出！
