　　“叶先生，多谢您救了我！不过我还是想问您一句，您为什么要帮我？”
<br/>　　两人往前走出数里，冥沧荨忽然站定，一双明亮的大眼睛，直勾勾的盯着叶尘，一脸好奇问道。
<br/>　　叶尘微微一笑，
<br/>　　“如果我说，我们两个的缘分，从前世就开始了，你信吗？”
<br/>　　冥沧荨先是微微一愣，继而两眼顿时飞快躲闪开来，不由得低下头，满脸羞色，低声道：
<br/>　　“叶先生，您说什么呢！”
<br/>　　说完这话，冥沧荨心中小鹿乱撞，跺了跺脚，然后满脸通红的跑开了。
<br/>　　“额……”
<br/>　　叶尘这才发现，自己这话有些歧义，这丫头多半是误会自己对她有意思，顿时一阵无语。
<br/>　　叶尘追上去，正准备好好解释一番。
<br/>　　就在这时，
<br/>　　前方远处，突然传来一道惊人的嘶吼声，犹如远古凶兽，令人为之心悸。
<br/>　　叶尘顿时一喜，
<br/>　　“应该是灭杀仙君的遗迹要出世了！走，我们也去看看！”
<br/>　　叶尘高兴之余，也没有多想，拉起冥沧荨的皓腕，便朝那道恐怖声音的方向，飞驰了过去。
<br/>　　冥沧荨原本就误会了叶尘，此时小手被叶尘紧紧抓着，顿时芳心一阵乱颤，不由得瞥了一眼叶尘俊朗的侧脸，一时间都有些痴了，浑然不知道自己身在何方。
<br/>　　一转眼，
<br/>　　两人已经跨越上百里的距离，来到一处草木异常旺盛的山谷之中。
<br/>　　“什么人？”
<br/>　　两人才刚刚踏入上古，一道暴喝声响起，
<br/>　　继而，
<br/>　　唰唰唰！
<br/>　　数十道身影，陡然出现在周围，将两人围在中央。
<br/>　　“嗯？”
<br/>　　叶尘大眼一扫，顿时微微一惊，
<br/>　　眼前这些人之中，居然足足有五位化神天君！
<br/>　　而且其中两人的修为，甚至比之前的冥家大长老还有强上一些。
<br/>　　冥沧荨看到这些人，顿时从方才的甜蜜中回过神来，连忙低声道：
<br/>　　“叶大哥，这些是其他三大家族的人，交给我来解决吧！”
<br/>　　叶尘顿时恍然，点了点头，微微退了一步。
<br/>　　通过他之前的了解，整个南斗星域，有四大家族，虽然他们彼此是竞争关系，但是表面上却还维持着和平，轻易不会动手。
<br/>　　果不其然，
<br/>　　对方这些人中，很快也有人认出了冥沧荨，
<br/>　　“呵呵，我当是谁？这不是冥家的那位天之娇女，冥沧荨吗？”
<br/>　　说话的是一名青年男子，看着比冥沧荨要大上几岁，应该是元婴境五重的修为。
<br/>　　“他叫雷洛风，是雷家的年轻一辈中的天才人物……”
<br/>　　冥沧荨低声快速向叶尘介绍了一番之后，直接迈步上前，冲着其他家族的那无名化神强者一一行了一礼，
<br/>　　“晚辈冥家，冥沧荨，拜见诸位前辈！”
<br/>　　“嗯！”
<br/>　　其中一名老者点了点头，脸色露出一抹奇怪之色，道：
<br/>　　“原来是冥家的丫头啊！你们族中的长辈呢？怎么只有你一个人？”
<br/>　　冥沧荨虽然单纯，却也一点也不傻，自然不可能将之前的事情告诉外人，于是道：
<br/>　　“晚辈和族中长辈在途中走散了，他们应该很快就会赶过来了！”
<br/>　　说到这里，冥沧荨话锋一转，转移话题道：
<br/>　　“对了，这个山谷，应该就是进入灭世仙君遗迹的入口吧？诸位前辈为何不进去？”
<br/>　　没等那老者开口，一旁的那个雷洛风冷冷一笑，道：
<br/>　　“不怕死的话，你就进去啊，没人拦着你！”
<br/>　　冥沧荨眉头微微一皱，她和雷洛风同为南斗星域的天才人物，雷洛风因为一直被她压着一头，所以一直对她有敌意，她也懒得理会，向其他家族一番打听之后，总算知道了事情的来龙去脉，
<br/>　　“擎苍宗的韩月公子，突然驾临此地！而且已经放出话来，在他没有出来之前，任何人不得踏入谷中一步！”
<br/>　　“擎苍宗？您说的是八大仙门之一的那个擎苍宗吗？”
<br/>　　冥沧荨听到这个消息，也是大为震惊。
<br/>　　八大仙门！
<br/>　　对于他们这些南斗星域四大家族来说，实在是太过遥远，简直犹如大象和蝼蚁，两者有着天壤之别！
<br/>　　一旁的雷洛风，再次冷笑道：
<br/>　　“在中央大星域，还有第二个擎苍宗吗？据说这位韩月公子，和我们差不多年纪，如今都已经是半步化神境界了呢！冥沧荨，你也就比我的修炼天赋高一点，和真正的天才相比，你还差得远呢！”
<br/>　　雷洛风好不容易逮到机会，便不失时机的冷嘲热讽起来。
<br/>　　冥沧荨眉头微微一皱，正要开口回击，就在这个时候，一旁的叶尘，忽然直接朝谷中走去。
<br/>　　冥沧荨见状，顿时吓了一跳，慌忙上前一把拉住叶尘，
<br/>　　“叶大哥，你做什么？”
<br/>　　叶尘直接道：
<br/>　　“我想进去看看！”
<br/>　　就在刚刚，他忽然感应到，眼前的山谷之中，隐隐有一股充满毁灭的气息。
<br/>　　如果他所料不错，这道气息，应该就是那把传说中的灭世魔刀！
<br/>　　冥沧荨听到这话，顿时两眼一瞪，
<br/>　　“叶大哥，你疯了？你没听他们说吗？擎苍宗的韩月公子在里面，你这个时候不能进去！”
<br/>　　叶尘淡淡一笑，
<br/>　　“韩月公子？废物一个罢了！我还没有放在眼里！”
<br/>　　轰！
<br/>　　叶尘这话一出口，周围顿时死一般的寂静，继而全都一副看傻子的表情望向叶尘。
<br/>　　韩月公子是何许人？
<br/>　　那可是堂堂八大仙门之一擎苍宗的绝世天才！
<br/>　　随便抛出一句话，便可以让他们这些南斗星域三大家族的强者，全都不敢越雷池一步。
<br/>　　先抛开他背后的擎苍宗不说，即便是韩月公子本身，那也是半步化神境的强者！
<br/>　　再加上强大的功法和法宝，即便是寻常化神中期的强者，恐怕都不是他的对手！
<br/>　　这家伙是什么来头，竟然敢说韩月公子是废物？
<br/>　　“叶大哥！这种话……不可以乱说的！”
<br/>　　冥沧荨反应过来之后，顿时俏脸发白，娇躯都忍不住颤抖起来。
<br/>　　纵然叶尘之前的表现，的确让他感觉无比惊艳，在冥沧荨看来，叶尘的实力，未必比那位韩月公子差，甚至还犹有过之。
<br/>　　可是，那韩月公子的背后，可是擎苍宗！是八大仙门之一啊！
<br/>　　要知道，八大仙门，那可是整个修真界最最顶级的势力！
